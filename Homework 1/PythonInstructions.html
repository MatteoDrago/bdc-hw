Instruction for using Apache Spark with Python

Windows

  1. First of all you need to install Python and Spark on your computer. In order to install Spark, follow the instructions on this link, while for a good configuration of Python use the following;
  2. Download and install Anaconda (select the version of Python that you prefer): this is a package which comes with the most useful modules that you will need for your scripts. Moreover, this will automatically install on your PC Jupyter Notebook, which is another user friendly IDE to run Python Scripts.
  3. After that, you need to open an Anaconda Prompt on the folder where you want to save your scripts. To install pyspark you have two options: either you create a dedicated environment inside Anaconda, which will be the only place where you can use pyspark, or you install it globally. The latter option is the most simple and is done by typing on the Anaconda Prompt simply: 

    pip install pyspark

    We refer to the Ananconda documentation for all the steps to create dedicated environments.
  4. Last step consist of importing the modules that you need on your script using the instruction 

    from pyspark import [...]

    and this works both in a jupyter notebook and in common .py file. Be aware that you need to run your .py script from the anaconda prompt if you haven't choose anaconda as you're default python interpreter.
